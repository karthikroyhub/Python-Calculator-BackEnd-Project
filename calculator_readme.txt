# Python Calculator

A command-line calculator built using Python. This project demonstrates core programming concepts such as control flow, exception handling, and the use of `match-case`.

---

## Features

* Addition
* Subtraction
* Multiplication
* Division (with zero-division handling)
* Continuous execution using loop
* All Clear (AC) functionality
* Exception handling for invalid inputs
* Implemented using `match-case` (Python 3.10+)

---

## Technologies Used

* Python 3.x
* Visual Studio Code
* Git and GitHub

## Usage

* Select an operation from the menu
* Enter two numbers separated by space
* View the result
* Use "AC" to reset
* Use "Exit" to close the program

---

## Error Handling

* Handles non-numeric input
* Prevents division by zero
* Validates user choices
